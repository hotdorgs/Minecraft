Site HotDorgs Modpack Minecraft
====================================

Este site foi criado para facilitar o download e instalação do modpack de Minecraft criado por HotDorgs para seus amigos e comunidade.

Funcionalidades:
- Instruções para instalar o modpack tanto no Minecraft original quanto pirata (com vídeos).
- Botão para copiar o código do modpack para uso no Curse Forge.
- Links para YouTube, Discord e Instagram do HotDorgs.
- Visual moderno, colorido e inspirado na skin e logo do criador.
- Direitos autorais destacados no rodapé.

Como usar/editar:
- Edite o arquivo index.html para alterar textos, links ou o código do modpack.
- Edite styles.css para mudar as cores ou o visual.
- Substitua o arquivo 'hot dorgs.png' pela logo desejada.

Publicação:
- Basta subir os arquivos para qualquer hospedagem estática (Netlify, Vercel, GitHub Pages, etc).
